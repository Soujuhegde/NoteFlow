# NoteFlow 📝
NoteFlow is a secure online note editor that lets you create, edit, and manage notes with **real-time sync** and **cloud storage**.  
It preserves rich text formatting and assigns unique secret IDs for easy access.  
Optimized for desktop and mobile (best on desktop site), NoteFlow offers a seamless, user-friendly experience.



## 🚀 Features
- Intuitive UI for quick note creation, renaming, and deletion  
- Secure cloud storage with data privacy  
- Real-time auto-save and synchronization  
- Unique secret ID for each note for easy retrieval and sharing  
- Supports rich text and font formatting preservation  
- Responsive design optimized for desktop and mobile (best on desktop site)  


## 🛠️ Tech Stack
- **Frontend:** HTML, CSS (Tailwind CSS), JavaScript (with AJAX for async operations)  
- **Backend:** PHP  
- **Database:** MySQL  
- **Modern Web Practices:**
  - Utility-first styling with **Tailwind CSS**  
  - **AJAX-based real-time sync** for auto-save and smooth user experience  
  - **Cloud-ready deployment** (XAMPP/WAMP locally, can be extended to AWS, Azure, or Heroku)
